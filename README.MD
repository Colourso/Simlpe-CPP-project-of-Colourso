## RetroSnaker_Colourso

一个简单的贪吃蛇小游戏。

VS2017 + [Easyx](https://easyx.cn/)制作。

Easyx是针对 C++ 的图形库，可以帮助 C 语言初学者快速上手图形和游戏编程。其官网( https://easyx.cn/ )有下载以及安装教程和使用范例。
